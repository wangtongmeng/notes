# 真机调试（安卓+Chrome）
## 流程
- 手机电脑通过数据线连接好
- 打开手机的【设置】选项,找到并进入【关于手机】，连续5-7次点击【版本号】开启开发者模式
- 返回手机【设置】页,搜索开发者
- 选择usb配置 rndis方式
- 勾选并允许USB调试
- 打开手机浏览器
- 打开谷歌调试网址：chrome://inspect/#devices
- 找到手机上的网址，点击Inspect
