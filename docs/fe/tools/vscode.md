# vscode

## vscode 插件

1. **`Import Cost`**
   - **作用**: 在你导入模块（`import` 或 `require`）时，直接在编辑器中显示该模块的压缩大小和 gzip 大小，非常直观。
   - **如何使用**:
     1. 打开 VS Code 扩展商店 (Ctrl+Shift+X)。
     2. 搜索 `Import Cost`。
     3. 点击安装。
     4. 安装后，在代码中尝试导入一个包，就会看到大小提示。
2. **`npm Intellisense`** (及类似查看包详情的插件)
   - **作用**: 当你在 `package.json` 中写包名或在代码中导入时，悬停在包名上，会弹出信息框，显示包的描述、入口文件、当前/最新版本、**压缩后代码体积**、预估下载时间、依赖数等。
   - **如何使用**:
     1. 打开 VS Code 扩展商店。
     2. 搜索 `npm Intellisense`。
     3. 点击安装。
     4. 在 `package.json` 里输入包名，或者在代码里 `import { } from '...'` 时，把光标放在包名上，就会显示详情。
