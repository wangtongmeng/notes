# nuxt.js 实现 ssr

## 安装

```bash
npx nuxi@latest init nuxt3-learn
# 选择 npm
cd nuxt3-learn
npm run dev
```





基本原理

- 通过 Vue 的 server-renderer 模块将 Vue 应用实例转换成一段文本的 HTML 字符串；
- 通过 Nodejs 创建一个静态 Web 服务器；
- 通过 Nodejs 将服务端所转好的 HTML 结构发送给浏览器端进行展示。
