## 1.引言

MySQL 提供了丰富的内置函数，可以帮助我们更高效地处理数据。这些函数涵盖了字符串处理、数值计算、日期时间操作等多个方面。

本文将从最常用的函数开始，逐步介绍字符串函数、数字函数、日期函数和高级函数的使用方法。通过实际示例，你将学会如何在查询中使用这些函数来处理和转换数据。

## 2.函数基础概念

### 2.1.什么是函数？

函数是一段预定义的代码，可以接收输入参数，执行特定操作，并返回结果。在 MySQL 中，函数可以直接在 SQL 语句中使用。

**函数的基本语法**：


```sql
函数名(参数1, 参数2, ...)
```

**示例**：


```sql
SELECT UPPER('hello');
-- 返回：HELLO
```

### 2.2.函数的使用场景

函数可以在以下场景中使用：

- **SELECT 语句**：在查询结果中处理数据
- **WHERE 子句**：在条件判断中使用
- **ORDER BY 子句**：在排序中使用
- **GROUP BY 子句**：在分组中使用

## 3.字符串函数

字符串函数用于处理文本数据，包括字符串连接、截取、替换、大小写转换等操作。

### 3.1.字符串长度函数

**CHAR_LENGTH()** 和 **CHARACTER_LENGTH()** 用于获取字符串的字符数：


```sql
SELECT CHAR_LENGTH('数据库') AS 字符数;
-- 返回：3

SELECT CHARACTER_LENGTH('MySQL教程') AS 长度;
-- 返回：7
```

**注意**：这两个函数计算的是字符数，不是字节数。对于中文等多字节字符，每个字符计为 1。

### 3.2.字符串连接函数

**CONCAT()** 用于将多个字符串连接成一个字符串：


```sql
SELECT CONCAT('MySQL', ' ', '数据库', ' ', '教程') AS 完整标题;
-- 返回：MySQL 数据库 教程
```

**CONCAT_WS()** 使用指定的分隔符连接字符串：


```sql
SELECT CONCAT_WS(' - ', '北京', '上海', '广州') AS 城市列表;
-- 返回：北京 - 上海 - 广州
```

### 3.3.字符串大小写转换

**UPPER()** 和 **UCASE()** 将字符串转换为大写：


```sql
SELECT UPPER('mysql database') AS 大写;
-- 返回：MYSQL DATABASE

SELECT UCASE('hello world') AS 大写;
-- 返回：HELLO WORLD
```

**LOWER()** 和 **LCASE()** 将字符串转换为小写：


```sql
SELECT LOWER('HELLO WORLD') AS 小写;
-- 返回：hello world
```

### 3.4.字符串截取函数

**LEFT()** 从左侧截取指定长度的字符：


```sql
SELECT LEFT('MySQL数据库', 5) AS 左侧字符;
-- 返回：MySQL
```

**RIGHT()** 从右侧截取指定长度的字符：


```sql
SELECT RIGHT('MySQL数据库', 3) AS 右侧字符;
-- 返回：数据库
```

**SUBSTRING()** 或 **SUBSTR()** 从指定位置截取指定长度的字符：


```sql
SELECT SUBSTRING('MySQL数据库教程', 6, 3) AS 截取结果;
-- 返回：数据库

SELECT SUBSTR('MySQL数据库教程', 1, 5) AS 截取结果;
-- 返回：MySQL
```

**语法说明**：

- `SUBSTRING(字符串, 起始位置, 长度)`
- 起始位置从 1 开始计数

### 3.5.字符串查找函数

**LOCATE()** 查找子字符串在字符串中的位置：


```sql
SELECT LOCATE('数据库', 'MySQL数据库教程') AS 位置;
-- 返回：6
```

**POSITION()** 也可以查找子字符串位置：


```sql
SELECT POSITION('SQL' IN 'MySQL数据库') AS 位置;
-- 返回：3
```

### 3.6.字符串替换函数

**REPLACE()** 替换字符串中的指定内容：


```sql
SELECT REPLACE('MySQL数据库', 'MySQL', 'PostgreSQL') AS 替换结果;
-- 返回：PostgreSQL数据库
```

**INSERT()** 在指定位置插入新字符串，替换原有字符：


```sql
SELECT INSERT('MySQL教程', 6, 2, '数据库') AS 插入结果;
-- 返回：MySQL数据库
```

**语法说明**：

- `INSERT(字符串, 起始位置, 替换长度, 新字符串)`

### 3.7.字符串去除空格

**TRIM()** 去除字符串首尾的空格：


```sql
SELECT TRIM('  MySQL数据库  ') AS 去除空格;
-- 返回：MySQL数据库
```

**LTRIM()** 去除左侧空格：


```sql
SELECT LTRIM('  MySQL数据库') AS 去除左侧空格;
-- 返回：MySQL数据库
```

**RTRIM()** 去除右侧空格：


```sql
SELECT RTRIM('MySQL数据库  ') AS 去除右侧空格;
-- 返回：MySQL数据库
```

### 3.8.字符串填充函数

**LPAD()** 在左侧填充字符，使字符串达到指定长度：


```sql
SELECT LPAD('123', 6, '0') AS 左侧填充;
-- 返回：000123
```

**RPAD()** 在右侧填充字符：


```sql
SELECT RPAD('123', 6, '0') AS 右侧填充;
-- 返回：123000
```

### 3.9.其他常用字符串函数

**REVERSE()** 反转字符串：


```sql
SELECT REVERSE('MySQL') AS 反转;
-- 返回：LQSyM
```

**REPEAT()** 重复字符串指定次数：


```sql
SELECT REPEAT('Hi', 3) AS 重复;
-- 返回：HiHiHi
```

**FORMAT()** 格式化数字，添加千位分隔符：


```sql
SELECT FORMAT(1234567.89, 2) AS 格式化;
-- 返回：1,234,567.89
```

## 4.数字函数

数字函数用于进行数学计算，包括四舍五入、取整、绝对值、幂运算等。

### 4.1.基本数学函数

**ABS()** 返回绝对值：


```sql
SELECT ABS(-15) AS 绝对值;
-- 返回：15
```

**ROUND()** 四舍五入到指定小数位：


```sql
SELECT ROUND(3.14159, 2) AS 四舍五入;
-- 返回：3.14

SELECT ROUND(3.6) AS 四舍五入;
-- 返回：4
```

**TRUNCATE()** 截断到指定小数位（不四舍五入）：


```sql
SELECT TRUNCATE(3.14159, 2) AS 截断;
-- 返回：3.14
```

### 4.2.取整函数

**CEIL()** 或 **CEILING()** 向上取整：


```sql
SELECT CEIL(3.2) AS 向上取整;
-- 返回：4

SELECT CEILING(3.7) AS 向上取整;
-- 返回：4
```

**FLOOR()** 向下取整：


```sql
SELECT FLOOR(3.7) AS 向下取整;
-- 返回：3
```

### 4.3.幂运算和开方

**POW()** 或 **POWER()** 计算幂：


```sql
SELECT POW(2, 3) AS 幂运算;
-- 返回：8

SELECT POWER(5, 2) AS 幂运算;
-- 返回：25
```

**SQRT()** 计算平方根：


```sql
SELECT SQRT(25) AS 平方根;
-- 返回：5
```

### 4.4.取余和整除

**MOD()** 计算余数：


```sql
SELECT MOD(10, 3) AS 余数;
-- 返回：1
```

**DIV** 进行整除运算：


```sql
SELECT 10 DIV 3 AS 整除;
-- 返回：3
```

### 4.5.聚合函数

**COUNT()** 统计记录数：


```sql
SELECT COUNT(*) AS 总记录数 FROM students;
```

**SUM()** 计算总和：


```sql
SELECT SUM(score) AS 总分 FROM exam_results;
```

**AVG()** 计算平均值：


```sql
SELECT AVG(score) AS 平均分 FROM exam_results;
```

**MAX()** 返回最大值：


```sql
SELECT MAX(score) AS 最高分 FROM exam_results;
```

**MIN()** 返回最小值：


```sql
SELECT MIN(score) AS 最低分 FROM exam_results;
```

### 4.6.随机数和符号函数

**RAND()** 生成 0 到 1 之间的随机数：


```sql
SELECT RAND() AS 随机数;
-- 返回：0.123456789（每次不同）
```

**SIGN()** 返回数字的符号：


```sql
SELECT SIGN(-10) AS 符号;
-- 返回：-1

SELECT SIGN(10) AS 符号;
-- 返回：1

SELECT SIGN(0) AS 符号;
-- 返回：0
```

### 4.7.最值函数

**GREATEST()** 返回参数列表中的最大值：


```sql
SELECT GREATEST(10, 25, 8, 30) AS 最大值;
-- 返回：30
```

**LEAST()** 返回参数列表中的最小值：


```sql
SELECT LEAST(10, 25, 8, 30) AS 最小值;
-- 返回：8
```

## 5.日期函数

日期函数用于处理日期和时间数据，包括获取当前日期时间、日期计算、日期格式化等。

### 5.1.获取当前日期时间

**NOW()** 返回当前日期和时间：


```sql
SELECT NOW() AS 当前时间;
-- 返回：2024-01-15 14:30:25
```

**CURDATE()** 或 **CURRENT_DATE()** 返回当前日期：


```sql
SELECT CURDATE() AS 当前日期;
-- 返回：2024-01-15
```

**CURTIME()** 或 **CURRENT_TIME()** 返回当前时间：


```sql
SELECT CURTIME() AS 当前时间;
-- 返回：14:30:25
```

### 5.2.日期提取函数

**YEAR()** 提取年份：


```sql
SELECT YEAR('2024-01-15') AS 年份;
-- 返回：2024
```

**MONTH()** 提取月份：


```sql
SELECT MONTH('2024-01-15') AS 月份;
-- 返回：1
```

**DAY()** 提取日期：


```sql
SELECT DAY('2024-01-15') AS 日期;
-- 返回：15
```

**HOUR()**、**MINUTE()**、**SECOND()** 提取时间部分：


```sql
SELECT HOUR('14:30:25') AS 小时;
-- 返回：14

SELECT MINUTE('14:30:25') AS 分钟;
-- 返回：30

SELECT SECOND('14:30:25') AS 秒;
-- 返回：25
```

### 5.3.日期计算函数

**DATE_ADD()** 在日期上添加时间间隔：


```sql
SELECT DATE_ADD('2024-01-15', INTERVAL 10 DAY) AS 加10天;
-- 返回：2024-01-25

SELECT DATE_ADD('2024-01-15 14:30:00', INTERVAL 2 HOUR) AS 加2小时;
-- 返回：2024-01-15 16:30:00
```

**DATE_SUB()** 从日期中减去时间间隔：


```sql
SELECT DATE_SUB('2024-01-15', INTERVAL 5 DAY) AS 减5天;
-- 返回：2024-01-10
```

**DATEDIFF()** 计算两个日期之间的天数差：


```sql
SELECT DATEDIFF('2024-01-20', '2024-01-15') AS 天数差;
-- 返回：5
```

### 5.4.日期格式化函数

**DATE_FORMAT()** 按指定格式显示日期：


```sql
SELECT DATE_FORMAT('2024-01-15 14:30:25', '%Y年%m月%d日') AS 格式化;
-- 返回：2024年01月15日

SELECT DATE_FORMAT(NOW(), '%Y-%m-%d %H:%i:%s') AS 格式化;
-- 返回：2024-01-15 14:30:25
```

**常用格式符**：

- `%Y`：四位年份
- `%m`：月份（01-12）
- `%d`：日期（01-31）
- `%H`：小时（00-23）
- `%i`：分钟（00-59）
- `%s`：秒（00-59）

**STR_TO_DATE()** 将字符串转换为日期：


```sql
SELECT STR_TO_DATE('2024年1月15日', '%Y年%m月%d日') AS 日期;
-- 返回：2024-01-15
```

### 5.5.其他日期函数

**DAYNAME()** 返回星期名称：


```sql
SELECT DAYNAME('2024-01-15') AS 星期;
-- 返回：Monday
```

**MONTHNAME()** 返回月份名称：


```sql
SELECT MONTHNAME('2024-01-15') AS 月份名;
-- 返回：January
```

**WEEK()** 返回日期是一年中的第几周：


```sql
SELECT WEEK('2024-01-15') AS 周数;
-- 返回：2
```

## 6.高级函数

高级函数包括条件判断、类型转换、空值处理等功能。

### 6.1.条件判断函数

**IF()** 根据条件返回不同值：


```sql
SELECT IF(score >= 60, '及格', '不及格') AS 结果 FROM exam_results;
```

**CASE** 语句实现多条件判断：


```sql
SELECT 
    CASE 
        WHEN score >= 90 THEN '优秀'
        WHEN score >= 80 THEN '良好'
        WHEN score >= 60 THEN '及格'
        ELSE '不及格'
    END AS 等级
FROM exam_results;
```

### 6.2.空值处理函数

**IFNULL()** 如果第一个参数为 NULL，返回第二个参数：


```sql
SELECT IFNULL(name, '未知') AS 姓名 FROM users;
```

**COALESCE()** 返回参数列表中第一个非 NULL 值：


```sql
SELECT COALESCE(nickname, realname, '未命名') AS 显示名称 FROM users;
```

**ISNULL()** 判断表达式是否为 NULL：


```sql
SELECT ISNULL(email) AS 是否为空 FROM users;
-- 返回：1 表示 NULL，0 表示非 NULL
```

### 6.3.类型转换函数

**CAST()** 转换数据类型：


```sql
SELECT CAST('123' AS UNSIGNED) AS 数字;
-- 返回：123

SELECT CAST('2024-01-15' AS DATE) AS 日期;
-- 返回：2024-01-15
```

**CONVERT()** 也可以进行类型转换：


```sql
SELECT CONVERT('123', UNSIGNED) AS 数字;
-- 返回：123
```

### 6.4.系统信息函数

**DATABASE()** 返回当前数据库名：


```sql
SELECT DATABASE() AS 当前数据库;
```

**USER()** 或 **CURRENT_USER()** 返回当前用户：


```sql
SELECT USER() AS 当前用户;
```

**VERSION()** 返回 MySQL 版本：


```sql
SELECT VERSION() AS MySQL版本;
```

**LAST_INSERT_ID()** 返回最后插入的 AUTO_INCREMENT 值：


```sql
SELECT LAST_INSERT_ID() AS 最后插入ID;
```

## 7.函数使用技巧

### 7.1.函数组合使用

函数可以嵌套使用，实现复杂的数据处理：


```sql
SELECT UPPER(LEFT(name, 3)) AS 前三个字符大写 FROM users;
```

### 7.2.在 WHERE 子句中使用函数

函数可以在 WHERE 条件中使用：


```sql
SELECT * FROM orders 
WHERE YEAR(order_date) = 2024;
```

### 7.3.在 ORDER BY 中使用函数

可以使用函数结果进行排序：


```sql
SELECT * FROM products 
ORDER BY LENGTH(product_name) DESC;
```

### 7.4.性能注意事项

- 在 WHERE 子句中对列使用函数可能会影响索引的使用
- 如果可能，尽量在应用层进行数据处理
- 对于大量数据，考虑使用存储过程或视图

## 9.更多字符串函数详解

### 9.1.ASCII 码函数

**ASCII()** 返回字符串第一个字符的 ASCII 码值：


```sql
SELECT ASCII('A') AS ASCII值;
```

## 13.MySQL 8.0 新增函数

MySQL 8.0 版本引入了一些新函数，主要用于 JSON 数据处理和窗口函数。

### 13.1.JSON 函数

**JSON_OBJECT()** 创建 JSON 对象：


```sql
SELECT JSON_OBJECT('name', '张三', 'age', 25, 'city', '北京') AS JSON对象;
-- 返回：{"name": "张三", "age": 25, "city": "北京"}
```

**JSON_ARRAY()** 创建 JSON 数组：


```sql
SELECT JSON_ARRAY('苹果', '香蕉', '橙子') AS JSON数组;
-- 返回：["苹果", "香蕉", "橙子"]
```

**JSON_EXTRACT()** 从 JSON 中提取值：


```sql
SELECT JSON_EXTRACT('{"name": "张三", "age": 25}', '$.name') AS 姓名;
-- 返回："张三"
```

**JSON_CONTAINS()** 检查 JSON 是否包含指定值：


```sql
SELECT JSON_CONTAINS('{"name": "张三", "age": 25}', '25', '$.age') AS 包含;
-- 返回：1（表示包含）
```

### 13.2.窗口函数

**ROW_NUMBER()** 为结果集中的每一行分配一个唯一的序号：


```sql
SELECT 
    ROW_NUMBER() OVER (ORDER BY score DESC) AS 排名,
    name,
    score
FROM students;
```

**RANK()** 为结果集中的每一行分配排名（相同值会有相同排名，并跳过后续排名）：


```sql
SELECT 
    RANK() OVER (ORDER BY score DESC) AS 排名,
    name,
    score
FROM students;
```

**注意**：窗口函数需要在 MySQL 8.0 及以上版本才能使用。

## 14.函数使用最佳实践

### 14.1.性能优化建议

1. **避免在 WHERE 子句中对列使用函数**：


   ```sql
   -- 不推荐（无法使用索引） SELECT * FROM orders WHERE YEAR(order_date) = 2024;  -- 推荐（可以使用索引） SELECT * FROM orders WHERE order_date >= '2024-01-01' AND order_date < '2025-01-01';
   ```

2. **合理使用函数嵌套**：


   ```sql
   -- 适度嵌套，避免过度复杂 SELECT UPPER(LEFT(name, 5)) AS 结果 FROM users;
   ```

3. **使用函数处理数据展示，而非数据存储**：

   - 存储时使用标准格式
   - 查询时使用函数格式化显示

### 14.2.常见错误避免

1. **字符串函数处理 NULL 值**：


   ```sql
   -- 如果 name 为 NULL，CONCAT 结果也为 NULL SELECT CONCAT('Hello ', name) FROM users;  -- 使用 IFNULL 处理 SELECT CONCAT('Hello ', IFNULL(name, 'Guest')) FROM users;
   ```

2. **日期函数时区问题**：

   - 注意 NOW() 和 SYSDATE() 的区别
   - NOW() 返回语句开始执行时的时间
   - SYSDATE() 返回函数执行时的时间

3. **数值精度问题**：


   ```sql
   -- ROUND 和 TRUNCATE 的区别 SELECT ROUND(3.145, 2); -- 返回：3.15（四舍五入） SELECT TRUNCATE(3.145, 2); -- 返回：3.14（截断）
   ```

## 15.总结

本文全面介绍了 MySQL 中常用的内置函数，包括：

1. **字符串函数**：用于处理文本数据，包括连接、截取、替换、格式化等
2. **数字函数**：用于数学计算，包括四舍五入、取整、幂运算、三角函数等
3. **日期函数**：用于处理日期和时间，包括获取当前时间、日期计算、格式化等
4. **高级函数**：包括条件判断、类型转换、空值处理、系统信息等
5. **MySQL 8.0 新函数**：包括 JSON 函数和窗口函数

**关键要点**：

- 函数可以在 SELECT、WHERE、ORDER BY 等子句中使用
- 函数可以嵌套使用，实现复杂的数据处理
- 注意函数对 NULL 值的处理方式
- 在 WHERE 子句中使用函数可能影响索引性能
- 合理使用函数可以提高查询效率和代码可读性

掌握这些函数的使用方法，能够让你在数据库操作中更加高效地处理各种数据需求。在实际使用中，建议根据具体场景选择合适的函数，并注意性能优化。
